باستخدامك لبوت Social Analyzer Bot (@Social_Media_tools_bot)، فإنك توافق على الالتزام بشروط الخدمة هذه. إذا كنت لا توافق على أي جزء من هذه الشروط، يرجى عدم استخدام البوت.

1. وصف الخدمة

يقدم البوت خدمات تحليل حسابات التواصل الاجتماعي (يوتيوب، تيك توك، انستقرام، فيسبوك، سناب شات) بما في ذلك:

  • 📺 تحليل قنوات يوتيوب (المشتركين، الفيديوهات، المشاهدات، أوقات النشر)
  • 📸 تحليل حسابات سناب شات (المتابعين، مشاهدات القصص، معدلات التفاعل) - عبر Snapchat Kit API
  • 🎵 تحليل حسابات تيك توك (الملف الشخصي، الإحصائيات، الفيديوهات) - قيد التطوير
  • 📘 تحليل حسابات فيسبوك - قيد التطوير
  • 📷 تحليل حسابات انستقرام - قيد التطوير
  • 📄 إنشاء صفحة بايو شخصية للمستخدمين المميزين
  • 🤖 توصيات ذكاء اصطناعي مخصصة عبر Google Gemini API

2. تكامل سناب شات (Snapchat Kit)

شروط خاصة باستخدام ميزة سناب شات:

  • ميزة تحليل سناب شات تتطلب موافقة صريحة منك عبر OAuth 2.0
  • نحن نحصل فقط على البيانات العامة للملف الشخصي وإحصائيات الأداء الأساسية
  • لا نصل أبداً إلى: الرسائل الخاصة، المحادثات، أو المحتوى غير العام
  • يمكنك إلغاء صلاحية التطبيق في أي وقت من إعدادات سناب شات
  • نلتزم بسياسات واستخدامات Snapchat Kit API

3. تكامل تيك توك (TikTok API)

شروط خاصة باستخدام ميزة تيك توك (قيد التطوير):

  • سيتم تطبيق نفس معايير الخصوصية والأمان عند اكتمال التكامل
  • نلتزم بسياسات واستخدامات TikTok API

4. حسابات المستخدمين

أنت مسؤول عن الحفاظ على سرية بيانات حسابك في تلجرام. جميع الأنشطة التي تحدث عبر حسابك هي مسؤوليتك وحدك.

5. الاستخدام المقبول

أنت توافق على استخدام البوت فقط للأغراض القانونية وبما يتوافق مع قوانين بلدك. لا يجوز استخدام البوت لنشر محتوى غير قانوني أو ضار، أو محاولة استخراج البيانات بطرق غير مصرح بها، أو محاولة اختراق أو تعطيل الخدمة.

6. خطط الاشتراك والدفع

يقدم البوت خطة مجانية وخطط اشتراك مميزة.

  • الخطة المجانية: 2 تحليل يوتيوب يومياً، 0 توصية AI
  • الخطة المميزة: تحليل غير محدود، توصيات AI، صفحة بايو شخصية، دعم أولوية
  • الدفع: يتم عبر نجوم Telegram أو التحويل البنكي
  • سياسة الاسترجاع: الاشتراكات المميزة غير قابلة للاسترجاع بعد التفعيل

7. حدود المسؤولية

لن يكون مطور البوت مسؤولاً عن أي أضرار مباشرة أو غير مباشرة ناتجة عن استخدام أو عدم القدرة على استخدام البوت، بما في ذلك فقدان البيانات أو الأرباح. يتم تقديم الخدمة "كما هي" دون أي ضمانات.

8. خدمات الطرف الثالث

يعتمد البوت على واجهات برمجة تطبيقات (APIs) من:

  • Google YouTube API v3 - لتحليل قنوات يوتيوب
  • Snapchat Kit API - لتحليل حسابات سناب شات
  • Google Gemini API - لتوصيات الذكاء الاصطناعي
  • Supabase - لتخزين البيانات

أنت توافق على الالتزام بشروط استخدام هذه الخدمات عند استخدام البوت.

9. التعديلات على الخدمة

نحتفظ بالحق في تعديل أو إيقاف الخدمة في أي وقت دون إشعار مسبق. سنبذل جهداً معقولاً لإخطار المستخدمين بأي تغييرات جوهرية.

10. التعديلات على الشروط

قد نقوم بتحديث شروط الخدمة هذه من وقت لآخر. سنقوم بإخطارك بأي تغييرات جوهرية عبر البوت أو عبر قناة التحديثات. استمرار استخدامك للبوت بعد التغييرات يعني موافقتك على الشروط المحدثة.

11. إنهاء الخدمة

نحتفظ بالحق في إنهاء أو تعليق وصولك إلى البوت فوراً، دون إشعار مسبق، لأي سبب كان، بما في ذلك انتهاك شروط الخدمة. يمكنك أيضاً حذف بياناتك وإلغاء اشتراكك في أي وقت.

12. قانون التحكيم

تخضع هذه الشروط وتفسر وفقًا لقوانين الجمهورية اليمنية. يتم حل أي نزاعات من خلال محاكم صنعاء.
